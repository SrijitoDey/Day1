1. Why do we use value attribute of <option> tag?
2. What is <input type="hidden"  />  ? Why do we use?
3. Diff. between  <input type="submit"  />   and <button></button>  ?

1-The value attribute specifies the value to be sent to a server when a form is submitted. The content between the opening <option> and closing </option> tags is what the browsers will display in a drop-down list.
2-The <input type="hidden"> defines a hidden input field. A hidden field lets web developers include data that cannot be seen or modified by users when a form is submitted.
3-Button don't have predefined functions, its a pair tag, can add images.Can't do them with submit.
